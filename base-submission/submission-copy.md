# Base Photo Passport

## App Name

Base Photo Passport

## Tagline

Mint your photo identity

## Description

Turn your photo into a polished Base identity NFT, upload it to IPFS, and mint it onchain in one simple flow.

Character count: 113

## Category

NFT / identity / creator tool

## Short Pitch

Base Photo Passport lets anyone turn a personal photo into a styled identity collectible, store the image and metadata on IPFS, and mint the NFT on Base.

## Production Environment Variables

- `PINATA_JWT`: server-side Pinata API token for IPFS uploads
- `NEXT_PUBLIC_PHOTO_PASSPORT_CONTRACT_ADDRESS`: deployed ERC-721 contract address
- `NEXT_PUBLIC_BUILDER_CODE`: Base Builder Code after the app is registered
- `NEXT_PUBLIC_BASE_APP_ID`: Base.dev app id for domain ownership verification
